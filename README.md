# Online Sales Analysis

## Opis projekta
Ovaj projekat modeluje sistem za upravljanje proizvodima i korpom kupca koristeći OOP principe u Pythonu.

## Klase

### Product
- name
- price
- quantity
- display_info()
- update_quantity()

### ProductManager
- add_product()
- display_products()
- total_inventory_value()
- remove_product()

### Cart
- add_to_cart()
- calculate_total()
- display_cart()
